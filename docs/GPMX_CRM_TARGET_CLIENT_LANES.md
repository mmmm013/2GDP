# GPMx CRM Target Client Lanes

GPMx does not speak to everyone the same way. The flagship routes each person into a client lane.

## Active Client Lanes

1. BUYER
- Gift buyers and personal senders.
- Offer: K-KUT HUG / TUG.
- Link: https://www.k-kut.com/mom

2. TESTER
- Friends, collaborators, early testers.
- Offer: clarity test and feedback loop.
- Link: https://www.k-kut.com/mom or https://www.k-kut.com/find

3. ARTIST
- Artists, vocalists, composers, songwriters.
- Offer: free interest review and selected 4PE review.
- Link: /kreator-intake

4. RIGHTS_HOLDER
- Publishers, catalog owners, estates, master/publishing controllers.
- Offer: controlled catalog evaluation.
- Link: /kreator-intake

5. BUSINESS_CLIENT
- Businesses, nonprofits, sponsors, civic organizations.
- Offer: campaign HUGs, custom K-KUT sets, 4PE PROMOTER.
- Link: /commercial

6. PARTNER_INVESTOR
- Economic development, funders, sponsors, strategic allies.
- Offer: invention proof, pilot, funding conversation.
- Link: /contact

7. LEGAL_IP
- Patent attorneys, clinics, IP advisors.
- Offer: provisional patent package and invention review.
- Link: /contact

8. SUPERVISOR_LICENSE
- Music supervisors, ad/film/TV/podcast buyers.
- Offer: emotional-moment search, PIX/K-KUT/mini-KUT licensing path.
- Link: /commercial

9. FAN_SUPPORTER
- Listeners, supporters, early believers.
- Offer: listen, follow, share, support.
- Link: /who
